## 1. 🧮 Calculator (计算器)
*   **路径**: `/Calculator`
*   **主要功能**: 一个基于控制台的简单计算器程序。它可以执行基本的加减法，也可以执行更复杂的科学计算（如幂运算）。
*   **核心知识点**:
    *   **接口 (Interfaces)**: 定义了 `ICalculator` 接口，规定了所有计算器必须具备的基本行为（如 `Add`, `Sub`）。

    *   **方法重写 (Method Overriding)**: 使用 `virtual` 和 `override` 关键字定制不同计算器的问候语 (`SayHello`)。
    *   **类与对象的解耦**: `Phone` 类也实现了 `ICalculator` 接口，展示了即使是完全不同的事物（手机），只要遵守了接口契约，也能被当做计算器使用。

## 2. 🏦 IBankProject (银行账户系统)
*   **路径**: `/IBankProject`
*   **主要功能**: 一个模拟银行账户管理的控制台程序。支持普通账户和 VIP 账户的存款、取款操作，并计算余额。
*   **核心知识点**:
    *   **封装 (Encapsulation)**: 使用 `public double Balance { get; protected set; }` 保护账户余额，防止外部随意修改，只能通过存款/取款方法改变。
    *   **接口设计**: `IBankAccount` 接口定义了银行账户的标准操作。
    *   **继承与扩展**: `VIPAccount` 继承自 `NormalAccount`，并重写了 `Deposit` 方法（VIP 存款有额外奖励）。
    *   **里氏替换原则 (LSP)**: 程序中可以用 `IBankAccount` 或 `NormalAccount` 类型的变量来引用 `VIPAccount` 对象，且程序逻辑保持正确。

    *   **属性保护**: 再次强化了 `protected set` 的使用，确保“油量”只能由载具自身内部逻辑修改。
